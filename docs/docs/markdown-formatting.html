Markdown Formatting

A guide to the supported Markdown syntax across CharDB

0

Overview

CharDB supports Markdown in several text fields across the platform. Markdown is a lightweight way to add formatting—like bold, italics, headings, and links—using simple text characters. You don't need to know HTML; just type naturally and add a few symbols to style your content.

Basic Text Formatting

Use simple characters around your text to make it bold, italic, or strikethrough.

What you type What you get Use for
**bold text** bold text Emphasis, names, important details
*italic text* italic text Character thoughts, species names, subtle emphasis
~~strikethrough~~ strikethrough Corrections, redacted text, humorous asides
`inline code` inline code Registry IDs, technical identifiers
What you type
**Luna** is a *graceful* fox with ~~brown~~ silver fur.
What you see

Luna is a graceful fox with brown silver fur.

2

Headings

Start a line with one or more # symbols to create headings. More # symbols mean smaller headings.

What you type Result
# Heading 1 Largest heading
## Heading 2 Medium heading
### Heading 3 Smaller heading
What you type
# Description A cheerful fox who loves adventure. ## Personality Brave but cautious. ### Fun Facts Allergic to pollen.
What you see

Description

A cheerful fox who loves adventure.

Personality

Brave but cautious.

Fun Facts

Allergic to pollen.

Tip: Headings are especially useful for organizing long character details into sections like Description, Personality, and Backstory.
3

Links

Create clickable links using square brackets for the display text and parentheses for the URL.

What you type What you get
[Link text](https://example.com) A clickable link labeled "Link text"
What you type
Art by [StarryPaws](https://example.com/starrypaws)
What you see

Art by StarryPaws

Safety: All links in markdown fields open in a new tab and include security attributes to protect you. You'll never be navigated away from CharDB unexpectedly.
4

Lists

Create bullet lists with - or *, and numbered lists with 1., 2., etc.

What you type
**Likes:** - Warm blankets - Stargazing - Peanut butter **Top 3 skills:** 1. Stealth 2. Cooking 3. Navigation
What you see

Likes:

  • Warm blankets
  • Stargazing
  • Peanut butter

Top 3 skills:

  1. Stealth
  2. Cooking
  3. Navigation
5

Blockquotes

Start a line with > to create a blockquote. These are great for character quotes, lore excerpts, or any text you want to visually set apart.

What you type
> "The stars remember what we forget." > — Luna, gazing at the night sky
What you see

"The stars remember what we forget."
— Luna, gazing at the night sky

6

Tables

Create tables using pipes (|) and dashes (---). Tables are useful for stat blocks, comparisons, and structured character data.

What you type
| Stat | Value | | --- | --- | | Strength | 8 | | Agility | 14 | | Charisma | 12 |
What you see
StatValue
Strength8
Agility14
Charisma12

Using the Editor

Wherever CharDB supports markdown, you'll see a dedicated editor with these features:

  • Toolbar – A bar at the top with an Edit/Preview toggle and a quick-reference of supported syntax
  • Edit mode – A monospace text area where you type your markdown
  • Preview mode – Click the Preview button to see your formatted text exactly as others will see it
  • Character count – A footer showing how many characters you've used out of the maximum allowed
Tip: Get into the habit of clicking Preview before saving. It only takes a second and helps you catch formatting mistakes early.
